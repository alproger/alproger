## 🚀 Featured Projects

### IELTS Writing Checker API

An open-source IELTS Writing Task 2 checker built with FastAPI and OpenAI API.

- Checks IELTS Task 2 essays
- Gives estimated band score and detailed feedback
- Provides grammar, vocabulary, and improvement suggestions
- Includes local web interface and REST API

**Tech:** Python · FastAPI · OpenAI API · HTML · CSS · JavaScript · Docker

---

### Open Dataset API Platform

A backend API platform for serving public dataset records with search, filtering, sorting, pagination, and statistics.

- REST API for dataset records
- Search, filter, sort, and pagination
- API key protection for write operations
- Swagger documentation and automated tests

**Tech:** Python · FastAPI · SQLAlchemy · SQLite · Docker · Pytest

---

### School Canteen Management System

A Flask-based web application for school canteen meal ratings and daily statistics.

- Meal rating system
- Daily vote tracking
- Average rating calculation
- Admin panel
- Student of the Day feature

**Tech:** Python · Flask · SQLite · HTML · CSS · JavaScript

---

### Telegram Translator Bot

A simple Telegram bot that translates user messages into multiple languages.

- Telegram bot message handling
- Inline language buttons
- Translation API integration
- Multi-language support

**Tech:** Python · Telegram Bot API · Google Translate API
